# Happy Birthday Website

## Live Website

A simple static website created to celebrate a birthday with a special greeting and festive design.

## About

This project is a lightweight static webpage that displays a birthday greeting. It is designed to be simple, accessible, and easy to share with friends, family, or loved ones on their special day.

## Features

- Birthday-themed design
- Responsive and lightweight
- Easy to access through any modern web browser
- Hosted using GitHub Pages

## Usage

Simply open the link above and enjoy the birthday greeting.
